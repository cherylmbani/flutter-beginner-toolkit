# Hello Flutter App

## Description
A simple Flutter app that demonstrates **basic interactivity and UI customization**.  

- Displays text: **“Hello Flutter!”**  
- Pressing the button changes the text to **“Flutter is interactive!”**, makes it **blue**, and increases the font size.  
- Built as a beginner-friendly example for learning Flutter.

---

## Installation

### Prerequisites
- Flutter SDK installed  
- VS Code (or any code editor)  
- Chrome browser  

### Steps
1. Clone the repository:

```bash
git clone <your-repo-url>
cd flutter_beginner_toolkit
```
2. Run the app in Chrome:
```bash
flutter run -d chrome
```
3. Open lib/main.dart to review or edit the code.

### Usage
- Press the “Press Me” button to toggle the text.

- Observe the text color and size changing dynamically.

### Expected Output
- Initial state: “Hello Flutter!” in black, font size 24, button below labeled Press Me.

- After pressing the button: Text changes to “Flutter is interactive!”, color becomes blue, font size 32. Pressing again toggles back.
